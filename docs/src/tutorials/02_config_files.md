# Configuration files

This executable tutorial is maintained as a Jupyter notebook:

[`docs/src/notebooks/02_config_files.ipynb`](../notebooks/02_config_files.ipynb)

It uses real YAML files from:

[`docs/src/examples/configs/`](../examples/configs/index.md)

It covers:

- loading config text from a `.yml` file,
- expanding sweep/range/random-uniform tags,
- resolving references,
- resolving registry-backed object references, and
- keeping notebook code Julia-first while keeping configuration file-backed.
