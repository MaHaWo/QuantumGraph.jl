# Flux training

This executable tutorial is maintained as a Jupyter notebook:

[`docs/src/notebooks/03_flux_training.ipynb`](../notebooks/03_flux_training.ipynb)

It covers:

- building a simple `Flux.Chain`,
- defining a loss and evaluator,
- constructing a `Trainer`,
- running `fit_trainer!`, and
- inspecting reports and checkpoints.
